# Calculator - Project

A responsive, web-based calculator application featuring light and dark themes, built with HTML, CSS, and JavaScript.

## Interactive Calculator with Theme Switching

- This repository contains a fully functional, browser-based calculator. Developed using standard front-end technologies, the application provides a seamless user experience with two distinct visual themes: a professional dark mode and a clean light mode. The theme can be toggled to suit user preferences, demonstrating a practical implementation of dynamic styling.

### Key Features:

- Dynamic Theming: Seamlessly switch between light and dark themes with a user-friendly toggle.
- Responsive Design: The interface adapts to various screen sizes, ensuring functionality on both desktop and mobile devices.
- Standard Operations: Supports basic arithmetic functions, including addition, subtraction, multiplication, and division.
- Clean UI: Designed with a modern, minimalist aesthetic for optimal usability.

### Technologies Used:
- HTML5: For the application's structure.
- CSS3: For styling, including the implementation of custom properties for theme management.
- JavaScript: For handling calculator logic and dynamic theme switching.
